### Prerequisites

What things you need to install the software and how to install them:
* Python3.6 or plus;

* numpy (v1.17.2)
```
pip install numpy
```
* OpenCV(cv2)(v4.1.1.26)
```
pip install opencv-python
pip install opencv-contrib-python
```
* imutils (v0.5.3)
```
pip install imutils
```
* shutil
```
pip install pytest-shutil
```
* sklearn (v0.21.3)
```
pip install scikit-learn
```
